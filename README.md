# 理性圣诞 · 人文与秩序的节日叙事

一个关于圣诞节历史、伦理与社会意义的深度探索。

## 项目概述

这是一个静态中文网站，从人文和理性视角探讨圣诞节的本质意义。项目摒弃了商业化的节日叙事，重新审视圣诞节作为文明时间仪式的核心价值。

## 特性

- 🌟 **纯净静态网站** - 无构建工具，原生 HTML/CSS/JS
- 📱 **响应式设计** - 完美适配移动端和桌面端
- ⚡ **性能优化** - GPU 加速动画，懒加载，减少运动支持
- ♿ **无障碍访问** - ARIA 标签，键盘导航，高对比度模式
- ❄️ **动态效果** - 雪花粒子系统，滚动动画，视差效果

## 本地开发

## 核心内容

网站分为七个模块，探讨圣诞节的多重维度：

1. **圣诞的三重本质** - 时间仪式、伦理实践、共同体确认
2. **时间与秩序** - 互动圆环展示四个维度
3. **圣诞并非神话** - 从历史演进看文明发展
4. **节日与现代社会** - 传统与现代的对比分析
5. **人文视角的圣诞** - 多学科角度解读
6. **冬夜语录** - 哲学性的思考片段
7. **行动与反思** - 如何过一个理性的圣诞节
